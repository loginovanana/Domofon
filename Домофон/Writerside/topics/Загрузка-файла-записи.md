# Загрузка файла записи

Получить файл записи с сервера.

На устройстве может храниться запись оповещения «по умолчанию». Ее нельзя удалить или заменить. На устройство можно загрузить дополнительную запись оповещения. Дополнительную запись оповещения можно удалить.

Файлы записей оповещения представляют собой WAV файл с данными следующего формата: PCM, 16-бит, 1 канал, частота 16КГц. Могут быть загружены и другие файлы WAV/PCM, но при воспроизведении будет выполнена конвертация до указанного формата. В целях экономии внутреннего пространства накопителя, рекомендуется использовать записи именно в указанном формате. 
<tip> Максимальный размер файла для загрузки - 1 МБ.</tip>


## Формат запроса


| <format style="" color="ForestGreen"> POST </format> | URI: `http://{{panel_address}}/api/v1.cgi/emergency_alert/upload/1` |
|:----------------------------------------------------:|---------------------------------------------------------------------|

<note>
В заголовке запроса указывается Content-Type multipart/form-data.
</note>

### Тело запроса

<tabs>
<tab title="JSON">

```JSON
<binary_data>
```
</tab>
</tabs>
<note>
<p>
В ответе запроса Загрузка файла записи

```<binary_data>```
означает, что вы получаете двоичные данные, которые содержат информацию о файле записи, поэтому в теле запроса указываются данные файла записи.
</p>
</note>

## Коды ответа: 

| Код | Причина          | Описание                                                  |
|-----|------------------|-----------------------------------------------------------|
| 200 | OK               | Успешный ответ.                                           |
| 400 | Wrong parameters | Неверные параметры в теле запроса, либо в адресе запроса. |
| 500 | Internal error   | Внутренняя ошибка.                                        |

Узнайте больше в разделе [Коды ошибок](Коды-ошибок.md).
