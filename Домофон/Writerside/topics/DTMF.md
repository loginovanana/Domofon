# DTMF

DTMF (Dual-Tone Multi-Frequency) - это стандарт для передачи тональных сигналов, используемых для ввода данных, например, для набора номера на телефонах.

## Чтение метода DTMF {id="1"}

Получить информацию о текущем методе передачи DTMF, используемом устройством.

### Формат запроса

| <format style="" color="Blue"> GET </format>     | URI: `http://{{panel_address}}/api/v1.cgi/intercom/dtmf/method` |
|--------------------------------------------------|-----------------------------------------------------------------|

### Формат ответа

###  <format style="" color="LawnGreen">200 OK</format> 
<tabs>
<tab title="JSON">

```JSON
{
 "dtmf_method" : "rfc2833"
}

```
</tab>
</tabs>

| Поле        | Описание                       | Тип данных |
|-------------|--------------------------------|------------|
| dtmf_method | Метод DTMF (rfc2833/sip-info). | string     |


## Установка метода DTMF {id="2"}

|------------------------------------------------------|-----------------------------------------------------------------|
