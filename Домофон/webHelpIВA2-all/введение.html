IP-Домофон. Внешний API 7.0.2 Help

Введение

Вы открыли документацию API версии 7.0.2.

Внешний API устройства IP-домофон (далее по тексту – домофон) предназначен для обращения с внешней стороны (например, через web-интерфейс) в целях настройки, диагностики и удаленного управления IP-домофоном.

Внешний API применяется для следующих задач:

  1. управление списком пользователей и правами удаленного доступа;

  2. чтение информации о версии ПО, аппаратной ревизии и модели домофона;

  3. чтение информации о серийном номере и MAC-адресе домофона;

  4. сброс к настройкам конфигурации, предусмотренным по умолчанию;

  5. импорт/экспорт файла с настройками конфигурации;

  6. обновление встроенного программного обеспечения домофона;

  7. управление системным журналом (настройки Syslog, чтение записей);

  8. настройка сетевого подключения (IP, маска, шлюз, DHCP, DNS);

  9. настройка VPN клиента;

  10. ping-диагностика сетевого подключения;

  11. настройка даты, времени, часового пояса, SNTP;

  12. настройка VoIP клиента и SIP;

  13. настройка параметров видеосъемки, OSD, видеотрансляции, RTSP;

  14. управление замками дверей (открытие/закрытие, время открытия двери);

  15. управление данными электронных ключей (RFID);

  16. управление данными кодов доступа;

  17. управление данными для вызовов (адресация, настройки громкости, длительность);

  18. диагностика линии координатно-матричной сети;

  19. настройка системных звуков (звуки, громкость);

  20. управление переходом в специальные режимы;

  21. настройка вывода информации на дисплей в режиме ожидания;

  22. настройка световых режимов работы домофона;

  23. управление оповещением населения.

API реализован способом RESTful API.

API принимает запросы и отвечает в формате application/json.

API использует следующие стандартные HTTP методы для CRUD операций:

HTTP Method

Описание

GET

Чтение данных

POST

Внесение новых данных

PATCH

Изменение существующих данных

DELETE

Удаление данных
