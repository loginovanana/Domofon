IP-Домофон. Внешний API 7.0.2 Help

Чтение настроек NTP

Формат запроса

GET

URI: http://{{panel_address}}/api/v1.cgi/network/ntp

Формат ответа

HTTP/1.1 200 OK - пример успешного ответа.

application/json>

{ "bDstEnabled": false, "bNtpEnabled": true, "iRefreshTime": 60, "sNtpServers": "ntp0.ntp-servers.net ntp1.ntp-servers.net ntp2.ntp-servers.net 0.ru.pool.ntp.org 1.ru.pool.ntp.org 2.ru.pool.ntp.org", "sTimeZone": "RussianStandardTime-3" }

Атрибут

Описание

Тип данных

bDstEnabled

Включение перехода на летнее время.

boolean

bNtpEnabled

Включение синхронизации с ntp.

boolean

iRefreshTime

Интервал синхронизации с ntp сервером, в минутах.

integer

sNtpServers

Адреса ntp серверов, без указания портов. Только адрес или IP, через пробел.

string

sTimeZone

Имя временной зоны (список зон можно получить в отдельном запросе).

string
