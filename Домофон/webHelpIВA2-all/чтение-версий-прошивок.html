Формат запроса

GET

HTTP/1.1 200 OK - пример успешного ответа.

application/json>

Описание

sCPUFirmwareVersion

Указывает версию прошивки центрального процессора (CPU).

string

sMCUFirmwareVersion

Указывает версию прошивки микроконтроллера (MCU).

string
