IP-Домофон. Внешний API 7.0.2 Help

Добавление/редактирование данных об одном/нескольких ключах

Формат запроса

POST

URI: http://{{panel_address}}/api/v1.cgi/intercom/rfids

Тело запроса

application/json>

{ "aKeys" : [ { "iFlatID": 22, "iKeyType": 0, "iSceneID": 0, "iLockAccess": 0, "sKeyUid": "ab752589" }, { "iFlatID": 0, "iKeyType": 0, "iSceneID": 0, "iLockAccess": 0, "sKeyUid": "fc125589" } ] }

Формат ответа

HTTP/1.1 200 OK - пример успешного ответа.

Атрибут

Описание

Тип данных

aKeys

Массив, представляющий список ключей.

array

iFlatID

Указывает на номер квартиры, к которой привязан ключ.

integer

iKeyType

Указывает на тип ключа (RFID ключ).

integer

iLockAccess

Определяет, какие двери или зоны в здании можно открыть этим ключом.

integer

iSceneID

Указывает на конкретный режим в квартире, который активируется при использовании ключа.

integer

sKeyUid

Уникальный идентификатор RFID ключа.

string
