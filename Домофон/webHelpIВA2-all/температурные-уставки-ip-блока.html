Температурные уставки IP-блока

Чтение температурных уставок IP-блока

Формат запроса

GET

URI: http://{{panel_address}}/api/v1.cgi/intercom/gconfig/ip_block

Формат ответа

HTTP/1.1 200 OK - пример успешного ответа.

application/json>

{ "iTemperatureHeatOff": -10, "iTemperatureHeatOn": -20, "iTemperatureIPBlockPowerOff": -15, "iTemperatureIPBlockPowerOn": -10 }

Атрибут

Описание

Тип данных

iTemperatureHeatOff

Температура, при которой отключается отопление.

integer

iTemperatureHeatOn

Температура, при которой включается отопление.

integer

iTemperatureIPBlockPowerOff

Температура, при которой отключается питание блока IP.

integer

iTemperatureIPBlockPowerOn

Температура, при которой включается питание блока IP.

integer

Редактирование температурных уставок IP-блока

Формат запроса

POST

URI: http://{{panel_address}}/api/v1.cgi/intercom/gconfig/ip_block

Тело запроса

application/json>

{ "oGConfigIPBlock" : { "iTemperatureHeatOn" : -20, "iTemperatureHeatOff" : -10, "iTemperatureIPBlockPowerOn" : -10, "iTemperatureIPBlockPowerOff" : -15 } }

Атрибут

Описание

Тип данных

oGConfigIPBlock

Массив, представляющий температурные уставки IP-блока.

array

iTemperatureHeatOff

Температура, при которой отключается отопление.

integer

iTemperatureHeatOn

Температура, при которой включается отопление.

integer

iTemperatureIPBlockPowerOff

Температура, при которой отключается питание блока IP.

integer

iTemperatureIPBlockPowerOn

Температура, при которой включается питание блока IP.

integer

Формат ответа

HTTP/1.1 200 OK - пример успешного ответа.
