Чтение информации о домофоне

Чтобы распознать устройство, нужно запросить информацию о нём.

Формат запроса

GET

URI: http://{{panel_address}}/api/v1.cgi/system/device-info

Формат ответа

HTTP/1.1 200 OK - пример успешного ответа.

application/json>

{ "id": 0, "name": "deviceName", "ro": "true", "value": "IP Videointercom" }, { "id": 1, "name": "serialNumber", "ro": "true", "value": "000000" }, { "id": 2, "name": "firmwareVersion", "ro": "true", "value": "V0.4.0 build 2024.02.05" }, { "id": 3, "name": "manufacturer", "ro": "true", "value": "Lartech" }

Атрибут

Описание

Тип данных

id

Уникальный идентификатор или номер, присвоенный домофону.

integer

name

Имя или название домофона, для обозначения в системе.

string

ro

Указывает, что описание устройства не может быть изменено пользователем в системе.

boolean

value

Содержательное описание устройства.

string
