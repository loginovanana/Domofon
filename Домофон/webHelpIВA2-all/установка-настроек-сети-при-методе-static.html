Метод static

Чтение настроек сети при методе static

Формат запроса

GET

URI: http://{{panel_address}}/api/v1.cgi/network/lan/static

Формат ответа

HTTP/1.1 200 OK - пример успешного ответа.

application/json>

{ "sDNS1": "192.168.251.62", "sDNS2": "8.8.8.8", "sV4Address": "172.24.1.50", "sV4Gateway": "172.24.1.49", "sV4Netmask": "255.255.255.240" }

Атрибут

Описание

Тип данных

sDNS1

Содержит IP-адрес первого DNS-сервера.

string

sDNS2

Содержит IP-адрес второго DNS-сервера.

string

sV4Address

Содержит IP-адрес устройства в формате IPv4.

string

sV4Netmask

Содержит маску подсети в формате IPv4.

string

sV4Gateway

Содержит информацию о IP-адресе шлюза LoRaWAN в формате IPv4.

string
