IP-адреса/маски при методе static

Чтение данных IP-адреса/маски при методе static

Формат запроса

GET

URI: http://{{panel_address}}/api/v1.cgi/network/lan/static/ip

Формат ответа

HTTP/1.1 200 OK - пример успешного ответа.

application/json>

{ "sV4Address": "172.24.1.50", "sV4Netmask": "255.255.255.240" }

Атрибут

Описание

Тип данных

sV4Address

Содержит IP-адрес стройства в формате IPv4.

string

sV4Netmask

Содержит маску подсети в формате IPv4.

string

Установка данных IP-адреса/маски при методе static

string

sV4Netmask
