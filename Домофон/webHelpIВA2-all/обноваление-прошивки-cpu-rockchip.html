Обновление прошивки CPU RockChip

Размер файла обновления до 110 МБ.

Формат запроса

POST

URI: http://{{panel_address}}/api/v1.cgi/firmware/cpu/update

Тело запроса

В заголовке запроса указывается Content-Type multipart/form-data.

В теле запроса содержится файл обновления .upd.

application/json>

<binary_data>

Домофон определяет совместимость содержащегося в теле запроса файла обновления .upd.

Формат ответа

HTTP/1.1 200 OK - пример успешного ответа.

Если данный файл не является файлом обновления для Rockchip, то будет возвращена ошибка, и установка такого файла не начнётся.

Пример ответа:

HTTP/1.1 400

{ "error": { "code": 400, "message": "Unsupported file" } }
