Чтение настроек syslog

Формат запроса

GET

URI: http://{{panel_address}}/api/v1.cgi/system/syslog/settings

Формат ответа

HTTP/1.1 200 OK - пример успешного ответа.

application/json>

{ "address" : "192.168.100.10:514", "enable" : true, "level" : "debug", "protocol" : "udp", "retention_time" : 6000 }

Атрибут

Описание

Тип данных

address

Адрес сервера (IP и порт; имя сервера пока не поддерживается).

string

enable

Включение/выключение отправки событий.

boolean

level

Уровень событий, выше которого события надо отправлять на сервер.

string

protocol

Протокол передачи данных - tcp, либо udp.

string

retention_time

Время хранения событий.

integer
