Чтение системного времени

Формат запроса

GET

URI: http://{{panel_address}}/api/v1.cgi/system/time

Формат ответа

HTTP/1.1 200 OK - пример успешного ответа.

application/json>

{ "time": "2017-08-04T12:06:32" }

Атрибут

Описание

Тип данных

time

Содержит дату и время, записанные в формате YYYY-MM-DDThh:mm:ss (год-месяц-день T часы:минуты:секунды).

string в формате ISO 8601
