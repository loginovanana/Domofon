Специальные режимы

Чтение состояния специальных режимов

Формат запроса

GET

URI: http://{{panel_address}}/api/v1.cgi/intercom/gconfig/special_modes

Формат ответа

HTTP/1.1 200 OK - пример успешного ответа.

application/json>

{ "iAllKeyOpenMode": 0, "iPermanentLockOpen": 0 }

Атрибут

Описание

Тип данных

iAllKeyOpenMode

Режим открытия двери для всех ключей.

integer

iPermanentLockOpen

Режим постоянного открытия замков.

integer

Переход в специальный режим

Формат запроса

POST

URI: http://{{panel_address}}/api/v1.cgi/intercom/gconfig/special_modes

Тело запроса

application/json>

{ "oGConfigSpecialModes" : { "iPermanentLockOpen" : 0, "iAllKeyOpenMode" : 0 } }

Атрибут

Описание

Тип данных

oGConfigSpecialModes

Массив, представляющий специальные режимы.

array

iPermanentLockOpen

Режим постоянного открытия замков.

integer

iAllKeyOpenMode

Режим открытия двери для всех ключей.

integer

Формат ответа

HTTP/1.1 200 OK - пример успешного ответа.
