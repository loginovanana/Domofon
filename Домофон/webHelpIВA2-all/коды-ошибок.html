Коды ошибок

400 Неверные параметры в теле запроса, либо в адресе запроса

{ "error": { "code": 400, "message": "Wrong parameters" } }

Параметр

Описание

error

Массив, представляющий ошибку.

code

Код ошибки.

message

Сообщение об ошибке.

501 Запрошен несуществующий ресурс

{ "error": { "code": 501, "message": "Not Implemented" } }

Параметр

Описание

error

Массив, представляющий ошибку.

code

Код ошибки.

message

Сообщение об ошибке.

Примеры

  • /intercom/appartmentz – код 501

  • /intercom/appartments/z2 – код 400

  • GET /intercom/appartments/0 – код 400 (квартиры 0 не может быть).

  • GET /intercom/appartments/101 – код 404 (если квартиры 101 нет).
